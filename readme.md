# Naked Resume

**Brutally honest resume analyzer with humanized rewrites and job matching.**

## What It Does

- Analyzes your resume against any job description
- Scores match percentage (0-100%)
- Rewrites your resume in natural, human voice
- Finds matching jobs on LinkedIn, Indeed, Glassdoor
- Gives honest, professional feedback

## How to Use

1. Upload your resume (PDF or Word)
2. Paste job description
3. Enter your location
4. Click "Analyze & Find Jobs"
5. Get score, feedback, rewritten resume, and job links

## Tech Stack

- Python
- Gradio
- OpenAI GPT-3.5
- PyPDF2 & python-docx
